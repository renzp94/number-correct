# 什么是@renzp/number-correct?

`@renzp/number-correct`是一款零依赖、基于竖式计算方式矫正js数字计算精度的库，不管计算的数有多大都不会丢失精度。

## 什么是竖式计算?

竖式计算就是数学中的`列竖式`。如：`123 + 456 = 579`

`加法(+)`
```js
  1 2 3
+ 4 5 6
--------
  5 7 9
```

`减法(-)`
```js
  4 5 6
- 1 2 3
--------
  3 3 3
```

`乘法(*)`
```js
                        1         2       3
  *                     2         4       5
  -------------------------------------------------------
                        500    +  100   + 15   =   615
                 4000 + 800    +  120          =  4920 +
         20000 + 4000 + 600                    = 24600 +
      ---------------------------------------------------
                                                 30135
```
